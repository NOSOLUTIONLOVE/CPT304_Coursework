## Coursework

本目录用于存放 **CPT304 课程通用材料**：流程说明、作业模板、报告与计划文档等（与具体项目无关）。

### 快速入口

- **课程流程与综述**：`coursework/CPT304_Coursework/`
- **课程文档中心（brainstorms / plans / reports / guides / templates）**：`coursework/docs/`

### 维护规则

- 这里的内容以 **课程要求与作业交付** 为中心，不承载项目源码。
- 若某文档是对某个项目的特定说明，优先放到该项目目录下（例如 `projects/<project>/docs/`）。
